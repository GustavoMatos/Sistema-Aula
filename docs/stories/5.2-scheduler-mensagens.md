## Fluxo

```
CRON (1 min)
    │
    ▼
Buscar scheduled_messages (pending, scheduled_for <= NOW)
    │
    ▼
Para cada mensagem:
    ├─ Lead no mesmo estágio? → Se não, SKIP
    ├─ Automação pausada? → Se sim, SKIP
    ├─ Horário permitido? → Se não, RESCHEDULE
    ├─ Processar variáveis {{nome}}
    ├─ Enviar via Evolution API
    ├─ Salvar em messages
    └─ Marcar como 'sent'
```
